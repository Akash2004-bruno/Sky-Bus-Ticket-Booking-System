# SKY BUS Booking System

**SKY BUS** is a Python-based console application for booking bus tickets. It allows users to select routes, pick-up and drop points, coach types, journey dates, and payment methods. It also calculates fare, distance, and offers discounts for groups or specific payment methods. Users can even book return tickets and provide feedback.

---

## **Features**
- View available bus stops.
- Select pick-up and drop points.
- Book tickets for multiple passengers (up to 55).
- Choose coach type: AC, Non-AC, Sleeper.
- Automatic distance calculation between start and end locations.
- Fare calculation based on coach type, distance, night travel, and optional breaks with hotel stay.
- Apply discounts for group bookings and specific payment methods.
- Return ticket booking with 10% discount.
- Generate booking ID and seat numbers automatically.
- Provide feedback and rating after the journey.
